# Domain Guard

Tasks are categorized into domains:

## UI
Frontend code — React, Vue, HTML/CSS, TypeScript/JavaScript

## Contract
Smart contracts or backend logic — Solidity, Rust, Cairo, Move, Vyper

## Integration
Wiring between UI and contracts — API calls, ABIs, event listeners, adapters

## Rules
- Work on one domain at a time
- Complete contract tasks before integration tasks
- Complete integration tasks before UI tasks (if they share a feature)
